常用工具

  • 【QTranslate】是一款轻量级的免费电子词典工具，该工具已集成各大在线翻译引擎实现便捷的实时翻译功能，可以用来翻译长句子以及短语、词组等，通过开启鼠标手势我们还可以进行网页划词翻译，一键多引擎在线翻译着实的高效实用。目前支持的翻译服务主要包括：Google 谷歌翻译、Bing 必应词典、Promt、巴比伦 Babylon、SDL FreeTranslation.com、俄罗斯 Yandex 以及国内的有道翻译、百度翻译等常用在线词典引擎。

    官方网站：https://quest-app.appspot.com （可能需要翻墙）
    百度云：https://pan.baidu.com/s/1HaLNtpYfADC2YT1Et0zQmw 密码：mvcb

  • 【天若OCR文字识别】是一款Windows平台上的图片文字识别工具。无需低效地人眼识别图片中的文字后再手动敲打出来，只需上传文字图片或选取屏幕文字区域即可快速获得可编辑的图中文字。它不仅有强大的识别功能，还有很多实用的功能等你来探索。

    官方网站：https://tianruoocr.cn/

  • 【朗读女】是一个非常简单实用的语音朗读软件。可以用它来朗读文本文件、小说、学习外语、语音报时、校对文章、读网页新闻等（凡是屏幕上可选中且能复制的文本它都能朗读），还可以用它来制作有声小说和学习资料音频等（它支持生成mp3与及wav格式的声音文件）。

    官方网站：http://www.443w.com/tts/

  • 【Everything】是一款极速电脑文件搜索神器，能迅速地对文件进行定位，与 Windows 内置的搜索不同，Everything可以显示计算机上的每个文件和文件夹。它体积小巧，界面简洁易用，快速建立索引，快速搜索，同时占用极低的系统资源，实时跟踪文件变化，并且还可以通过http或ftp形式分享搜索。

    官方网站：https://www.voidtools.com

  • 【Hasher】是一款文件校验工具，能校验文件的 MD5、CRC32、SHA-1、SHA3 等值，确定文件传输的可靠性和一致性，通过对比下载文件的校验和，判断下载的文件是否出错，特别是系统镜像文件的校验，能判断镜像文件是否下载完整、是否被人修改过等，尤其 Windows 8 RTM 在 MSDN 发布了，提醒大家一定比对好 ISO 文件的 SHA1 值。

    官方网站：http://www.den4b.com/products/hasher

  • 【ReNamer】是一款体积小巧、功能强大灵活的文件批量重命名工具，该程序支持包括插入、删除、移除、替换、前缀、后缀以及正则表达式等十四种重命名规则。只需要将待重命名的单个或者大量文件拖放到文件窗口，接着再指定一个重命名规则便可以轻松执行重命名操作。

    官方网站：http://www.den4b.com/products/renamer

  • 【Beyond Compare】 是一款专业级的文件夹和文件对比工具。使用它可以很方便地对比出两个文件夹或者文件的不同之处，相差的每一个字节用颜色加以表示，查看方便，支持多种规则对比，找出差异，合并或同步文件和文件夹，并生成报告。（收费软件）

    官方网站：http://www.scootersoftware.com/

  • 【软媒魔方】是一款集十八般电脑优化设置工具于一身的强大电脑系统优化工具，其中的每一款优化设置工具都可以单独进行使用，不喜欢的完全可以直接删除，只留对自己有用的。

    官方网站：https://mofang.ruanmei.com/

  • 【PanDownload】是一款由中国吾爱著名软件开发达人 Kiryuu 采用C++ 精心编写与设计制作的基于百度云 API 接口的百度网盘资源高速下载器，通过该工具解析获取直链后调用 Aria2 或者更多其他 API 线路进行下载，简洁的外观却不失其功能的强劲，旨在更快捷地获取到想要的文件。

    官方网站：http://pandownload.com/

  • 【护眼宝】是一款通过智能调节屏幕色温过滤蓝光的软件，能有效减少屏幕蓝光对眼睛的伤害，从而保护眼睛防止近视。晚上暗光时能使屏幕色调变得更暖。

    官方网站：http://www.huyanapp.com

  • 【eDiary日记本】是一款界面简单清新，支持皮肤切换，编辑功能强大，具有严格的数据加密机制的电子日记本，并具有简单合理的文档管理功能，为您轻松梳理日常信息。

    官方网站：http://www.haoxg.net/

  • 【PotPlayer】 是一款内置强大解码器的高清视频播放器，能够开启硬解模式，而且能够播放网络直播源。

    官方网站：http://potplayer.daum.net
    网盘：https://carrot.ctfile.com/dir/11269771-18303671-1b5f58/

  • 【TeleList】是一款快速制作 VLC 和 Potplayer 等播放器的 XSPF、DPL、M3U 等播放源格式的工具，并能对各种直播源格式进行转换。

    官方网站：https://guihet.com/tvlive-telelist.html

  • 【MusicTools】是一款无损音乐下载工具，它可以下载各大音乐平台的无损音乐。

    网盘：https://www.lanzous.com/b395448
